## 正式发布前剩余检查点

需要在本机 `.env` 中提供有效 `QQ_APP_ID`、`QQ_APP_SECRET`，并由 QQ 开放平台账号持有人完成沙箱/测试群配置。随后验证：

- Webhook 签名挑战、签名校验、事件接收与 ACK；
- 群聊 `帮助`、今日/本周/季度、搜索、详情、下次更新；
- 两名用户订阅同一番剧后合并 `@` 推送；
- 每日/每周计划、时区和停用；
- 429 重试、永久失败、unknown 不自动重发及管理员明确补发；
- 移除机器人或关闭主动消息后停止推送。

若该 QQ 账号只能使用兼容 WebSocket，再额外验证断线重连；新部署默认使用 Webhook。
